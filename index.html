Open model release Run deployment
  1. Sign in to GitHub as the repository owner.
  2. Open the models release and edit it.
  3. Attach your .ply, .splat, .ksplat, or .spz file.
  4. Save the release.
  5. Open the deployment workflow and click Run workflow.
  6. Refresh this dashboard, select the hosted model, then click Share.
